Some tips for viewing the map:
  • The numerical value represented by the colour is visible at a scale of 1:2000 and below. The street names will also render at this scale.
  • Viewing the map without the "Hamilton Street Railway" layer rendered makes the results of this calculation
The "transit count" number is calculated as follows:
  • The two input data sets come from Open Street Map and the Hamilton Street Railway's GTFS repository. The HSR is a bus-only transit network. The data used here applies to the schedule on September 5, 2025.
  • Using the QGIS GTFS-GO plug-in, the number of buses arriving at each stop during the selected 24-hour period was calculated.
  • For individual, randomly selected buildings, the bus stops within 400 metres radius were identified. (See "improvements" below for a discussion on why buildings were randomly selected.)
  • Using the native Network Analysis algorithm in QGIS, the walking distance from each building to each nearby bus stop was calculated.
  • For each stop, a weight was assigned, according to walking distance: stops within 300 m recieved a weight of 1, stops beyond 400 m recieved a weight of zero, and stops between 300 m and 400 m recieved a weight between 1 and 0 that decreased linearly with distance.
  • The total transit count for each building was calculated as a sum of weight times number of bus "visits" for all nearby bus stops.

Improvements to come:
  • Obviously, this calculation has only been done for a small subset of the houses in Hamilton. This is mostly driven by 1) long calculation times in the network analysis step, and 2) data size limits in the free QGIS Cloud service. I'd like to find a way to run the calculation in parallel, or more efficiently, to include the rest of the city.
  • For some reason, the Open Street Map data set does not include many buildings in East Hamilton. There are building and road data sets available from Statistics Canada that may be higher quality.
  • It's unclear how sensitive the above insights are to choices in the search distance parameter or weighting function parameters. Making a more efficient calculation will allow me to explore the affects of these choices.
