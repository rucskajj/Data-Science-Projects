Description for a Map of Public Transit Accessibility in Hamilton, ON

Map link

Joey Rucska

Last updated on 16 September, 2025.

The goal of this project is to map the walking accessibility of the public transit network in Hamilton, ON. A map was produced using QGIS, and is available through this link. The colour of each building tracks the frequency that buses arrive at bus stops within walking distance--bright yellow is high frequency, dark blue/purple is low frequency. I refer to this quantity as the "transit count", and describe how it is calculated in more detail below.

Some tips for interpreting, viewing, and navigating the map:
  • The numerical value represented by the colour is visible at a scale of 1:2000 and below. The street names will also render at this scale.
  • Viewing the map without the "Hamilton Street Railway" layer rendered makes the distribution of the transit count quantity more apparent.
  • Specific streets, neighbourhoods, and addresses can be located using the search bar at the top.

Data & Methods

The two input data sets come from Open Street Map and the Hamilton Street Railway's GTFS repository. The HSR is a bus-only transit network. The data used here applies to the schedule on September 5, 2025.

The "transit count" number is calculated as follows:
  • Using the QGIS GTFS-GO plugin, the number of buses arriving at each HSR bus stop over the course of the selected 24-hour period is calculated.
  • For individual, randomly selected buildings, the bus stops within a 400 metre radius are identified. (See "improvements" below for a discussion on why buildings were randomly selected.)
  • Using the native Network Analysis algorithm in QGIS, the shortest walking distance from each building to all nearby bus stops is calculated.
  • For each of the nearby stops, a weight was assigned, according to walking distance: stops within 300 m recieved a weight of 1, stops beyond 400 m recieved a weight of zero, and stops between 300 m and 400 m recieved a weight between 1 and 0 that decreased linearly with distance.
  • The total transit count for each building was calculated as a sum of weight multpilied by the number of bus "arrivals" for all nearby bus stops. (This number was rounded to the nearest integer for display on the map.)

Future improvements:

  • Obviously, it would be better to do this calculation for all of the houses in the map. The current random sampling is driven by 1) long calculation times in the network analysis step, and 2) data size limits in the free QGIS Cloud service. I'd like to find a way to run the calculation in parallel, and/or more efficiently, to include the rest of the city.
  • For some reason, the Open Street Map data set does not include many buildings in East Hamilton. Statistics Canada provided building and road data sets which may be higher quality.
  • It's unclear how sensitive the above insights are to choices in the search distance parameter or weighting function parameters. Making a more efficient calculation will allow me to explore the affects of these choices.
